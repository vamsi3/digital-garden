title: Chromium
---

##### Set as default Browser

- chrome://settings/defaultBrowser

##### macOS

- System Settings → Desktop & Dock → Default Web Browser → Select 'Chromium.app'

###### Shortcuts

- `⌘ + .` Stop loading the page.
- `⌘ + [1-9]` Jump directly to a specific tab.
- `⌘ + ⇧ + A` Search all open tabs.
- `⌘ + ⇧ + N` Open a new Incognito window.
- `⌘ + ⇧ + T` Reopen the last closed tab.
- `⌘ + ⌥ + ←` Go to the previous tab.
- `⌘ + ⌥ + →` Go to the next tab.
- `⌘ + ⌥ + L` Open the Downloads page.
- `⌘ + D` Bookmark the current page.
- `⌘ + F` Find text on the current page.
- `⌘ + L` Jump to the address bar.
- `⌘ + R` Reload the page.
- `⌘ + T` Open a new tab.
- `⌘ + W` Close the current tab.
- `⌘ + Y` Open your browsing history.
- `⌘+⇧+A` Search open tabs
